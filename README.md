# RoomSQL Example App

Android application built with Kotlin, Jetpack Compose, and Room Database.  
The app allows users to manage products and user accounts.

## Features
- Product management: Add, Edit, Delete products
- ProductCard UI for displaying products
- User registration, login, and profile management
- Navigation between multiple screens using NavController
- Basic UI with RoomSQLTheme (Theme customization planned)
- Data persistence using Room Database
- Async operations using Kotlin Coroutines and Flow
- JSON loading for initial product data via JsonLoader

## Technical Details
- Built with Kotlin and Jetpack Compose
- Room Database for local data storage
- ViewModel and ViewModelFactory for state management
- Navigation with NavHost and composable destinations
- Kotlin Coroutines and Flow for asynchronous tasks

## Status
- Core functionality implemented: Product CRUD operations
- ProductCard UI implemented
- Other screens (Settings, some UI elements) under development
- Improvements and additional features ongoing
